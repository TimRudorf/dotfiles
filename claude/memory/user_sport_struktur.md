**Abend-Hauptsession (mit Tilman, ~19:00 Uhr):**
- Mo: Fitnessstudio
- Di: Long Run
- Mi: Fitnessstudio
- Do: Intervalle
- Fr: Oberkörper- oder Unterkörpertraining (allein)
- Sa: Oberkörper- oder Unterkörpertraining (allein, Komplementär zu Fr)
- So: Ruhetag

**Vormittags-Zweitsession (optional, je nach Tagesgeschehen):**
- An Fitnessstudio-Tagen (Mo, Mi): kleine Laufrunde am Vormittag
- An Lauf-Tagen abends (Di, Do): Schulter-Arme-Tag am Vormittag
- An Fr/Sa: keine fixe Zweitsession definiert
- So: nichts (Ruhetag)

**Regel für Vormittags-Slot:**
- Optional — wir versuchen ihn umzusetzen, aber er fällt zuerst, wenn der Tag zu kurz wird oder Wichtigeres kommt
- Im Kalender als "tentative" / niedrige Prio, nicht als harter Block
- Hauptsession 19:00 ist Anker, die fällt nur in echten Ausnahmefällen

**How to apply:** Beim Wochenraster bauen — Hauptsession fix als Tilman-Slot 19:00–~21:00, Vormittag als optionalen Slot 1–1,5h. Bei Konflikten zuerst Vormittags-Slot streichen, nicht Hauptsession.
